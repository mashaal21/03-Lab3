Image Processing – Tasks 1 \& 2



Task 1 (Geometric):



Scaling: increases image size (more pixels, no new details)

Rotation: rotates the image (120°)

Shear: tilts the image shape



Task 2 (Intensity):



Negative: inverts brightness

Log: enhances dark regions

Power (Gamma): controls brightness/contrast (γ<1 brighter, γ>1 darker)
